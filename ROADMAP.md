# BeenThere — Roadmap

A living planning document. It reflects current, agreed scope only — a scaffold
to plan against, not a feature promise. Keep it current as scope evolves.

## ✅ Done

- Guest Try experience: upload/select photo → automatic polaroid → WOW moment →
  pinned on the wooden world map → micro-celebration → explore the wall.
- Interactive Guest Wall (zoom/pan, mobile-first) with polaroids on the map.
- GPS/EXIF handling with a manual place fallback (never 0,0).
- AI Vision layer operational: `recognize` Edge Function (Gemini
  `gemini-2.5-flash`), deployed and confirmed on real devices.
- `/liquid-factory` dedicated landing page.
- Production deploy on Vercel.

## 🔜 Next

- Reusable "demo memories" architecture (seed multiple example memories).
- Anonymous telemetry (confidence / decision / accepted — never the image).
- Prompt & confidence-threshold tuning on real photos.
- Sharing (wall link + social).
- Accounts & cloud sync (magic link → Supabase).

## 🌅 Future

- Print / high-res poster export of the wall.
- _To be defined together._
