# Design archive

Historical and design material for BeenThere — **not used by the production app**.
It is kept in the repository as a record of the product's visual evolution.

## Structure

- `concepts/` — early hero/wall concept renders and explorations.
- `references/` — visual references, proportions, logo/pin rules, QR codes.
- `mockups/` — UI mockups and the customer-journey / final-UI screens.

Nothing in this folder is imported by `src/` or served from `public/`. Runtime
assets used by the app live in `public/assets/`.
